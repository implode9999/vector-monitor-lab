# Asteroids audio samples

These local WAV files are user-supplied reference samples for the Asteroids reproduction mode. They are loaded at runtime by `src/game/AsteroidsAudio.ts` so the local demo can match the original cabinet-style sound events while tuning gameplay and beam rendering.

Do not treat these files as generated project assets. Downstream users should verify the rights for these reference WAVs before redistributing them in another package, commercial build, or public release.
